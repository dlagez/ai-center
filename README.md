# AI中台项目

## 项目简介

一个完整的企业级AI中台解决方案，提供统一的AI能力接入、管理、治理和运营平台。

## 核心特性

### 架构层次
- **门户层**: Web管理后台、应用广场、Prompt管理、知识库管理、监控看板
- **应用接入层**: Chat UI、SDK、OpenAPI、第三方系统集成（企业微信、钉钉、OA、ERP等）
- **网关治理层**: API网关、认证授权、限流熔断、计费、安全审查
- **Agent编排层**: 基于LangGraph的工作流编排、多Agent协作、状态机管理
- **Prompt工程层**: Prompt模板管理、版本控制、A/B测试、发布审批
- **模型工具层**: 统一LLM适配、Embedding、Tools调用、MCP集成
